-> Clonar el repo
    git clone https://github.com/ZCoffeeCore/Okaff-
    cd Okaff-

-> Raíz
    npm install

-> Backend
    cd backend
    npm install --legacy-peer-deps
    (crear el archivo .env.local con las credenciales)

-> Frontend
    cd frontend
    npm install
    (crear el archivo .env.local con las credenciales)
    
-> Correr el proyecto (desde la raíz)
    cd ..
    npm run dev
